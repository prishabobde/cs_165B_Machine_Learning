## Implemented Algorithms:

Voted Perceptron for binary classification using weighted hypothesis voting
Linear Support Vector Machine (SVM) trained via stochastic gradient descent with hinge loss and L2 regularization
K-Nearest Neighbors (KNN) using Euclidean distance with deterministic tie-breaking
AdaBoost (Boosting) with custom weak learners based on weighted class centroids

## Functionality:

Data loading and preprocessing, including feature normalization (zero mean, unit variance) and bias augmentation
Train/test splitting with randomized shuffling for robust evaluation
Model training, prediction, and accuracy-based evaluation
Hyperparameter experimentation (e.g., dataset size, regularization strength, number of neighbors, boosting rounds)

## Technical Highlights:

Implemented optimization routines such as gradient-based updates and iterative weight reweighting
Explored tradeoffs between linear models, instance-based learning, and ensemble methods
Designed modular, reusable code for experimentation and comparative analysis

## Tech Stack:

Python, NumPy, Pandas
